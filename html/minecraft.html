Minecraft es un juego de aventura y construcción con bloques. Explora mundos generados aleatoriamente y construye cosas maravillosas, desde una simple casa hasta un fastuoso castillo. Juega en modo creativo con recursos ilimitados o conoce el mundo a fondo con el modo supervivencia, fabricando armas y armaduras para deshacerte de peligrosas criaturas.

Desarollador: Mojang Studios

Requisitos del Sistema

Requisitos mínimos

Procesador: Intel Core i3 3210 3.2 GHz o AMD A8 7600 APU 3.1 GHz o Apple M1 o modelo equivalente

Tarjeta Gráfica: Intel HD Graphics 4000 | AMD Radeon R5

Almacenamiento: 200 MB

Red: No es necesario

Sistema Operativo: Windows 7 y posterior O macOS 10.14.5 Mojave y posterior. x64, x86

Memoria RAM: 2GB de RAM


Requisitos recomendados

Procesador: Intel Core i5-4690 3.5 GHz o AMD A10-7800 APU 3.5 GHz o Apple M1 o modelo equivalente

Tarjeta Gráfica: NVIDIA serie GeForce 700 o serie AMD Radeon Rx 200 (excepto conjuntos de chips integrados) con OpenGL 4.45

Almacenamiento: 1GB

Red: Conexión de banda ancha a internet

Sistema Operativo: Windows 10 y posterior O macOS 10.14.5 Mojave y posterior.

Memoria RAM: 4GB de RAM
